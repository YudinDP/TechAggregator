Парсер товаров

Поддерживаемые источники и примеры ссылок

Парсинг и обновление цен работает через API-источники: API Systems (WB/Яндекс.Маркет), PricesAPI.io, eBay Browse API. По возможности используйте URL карточки; если URL нет — можно ввести только название.

  • Wildberries: https://www.wildberries.ru/catalog/123456/detail.aspx или https://wb.ru/catalog/123456/detail.aspx
  • Яндекс.Маркет: https://market.yandex.ru/card/123456/spec (и вариации /card/, /product--)
  • eBay: https://www.ebay.com/itm/123456789012 (требуются EBAY_CLIENT_ID/EBAY_CLIENT_SECRET в .env)
  • PricesAPI: поддерживает поиск по названию и по URL карточки магазина в поддерживаемых странах. По умолчанию используется country us (настраивается PRICESAPI_COUNTRY).

Изменение истории цен

Автообновление цен по ссылкам

Источники: API Systems, eBay, PricesAPI.io. Цены в рублях. Автообновление каталога — не чаще 1 раза в 3 дня; для товара ниже — вручную в любой момент.

Подробнее об обновлении

Учитываются только сохранённые у товара строки цен и ссылки. Для eBay/PricesAPI — по названию или по сохранённым ссылкам на поддерживаемые площадки.
